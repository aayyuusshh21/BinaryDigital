# To Run Locally
## Clone the project
```bash
  git clone https://github.com/aayyuusshh21/BinaryDigital.git
```

Go to the project directory

```bash
  cd binarydigital
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  json-server --watch db.json --port 3000
```

```bash
  npm run start
```
